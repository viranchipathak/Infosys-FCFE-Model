# Infosys FCFE Equity Valuation Model

This project presents a complete **Free Cash Flow to Equity (FCFE) valuation** of Infosys built using Microsoft Excel financial modelling.

The objective is to estimate the **intrinsic equity value per share** using discounted cash flow techniques commonly applied in:

- Equity Research  
- Corporate Finance  
- FP&A and Valuation roles  

---

## Model Structure

### 1. Assumptions
- Cost of equity (discount rate)  
- Terminal growth rate  
- Shares outstanding  
- Forecast period: 2025–2029  

### 2. FCFE Forecast & Discounting
- Five-year FCFE projections  
- Present value calculation using cost of equity  
- Terminal value using the perpetual growth method  

### 3. Valuation Summary & Sensitivity
- Total equity valuation  
- Intrinsic value per share  
- Two-variable sensitivity analysis  
  - Discount rate vs. terminal growth  

---

## Key Results

- **PV of FCFE (2025–2029):** ~₹131,691 crore  
- **PV of Terminal Value:** ~₹380,967 crore  
- **Total Equity Value:** ~₹5.13 lakh crore  
- **Intrinsic Value per Share:** ~₹1,235  

The Excel model reproduces the academic valuation and demonstrates the ability to convert **finance theory into practical financial modelling**.
